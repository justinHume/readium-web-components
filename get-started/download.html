Get Readium.js

Jul 27th, 2013

Versions

There are currently a few ways to get Readium.js.

SimpleReadium.js

SimpleReadium.js is a single javascript file that contains the entire set of Readium.js modules. However, it provides a single, reduced, API for simple applications, or getting used to using Readium.js in your application. SimpleReadium.js is used in the demo application.

SimpleReadium.js can be downloaded in minified gzip format (20KB), or as a zipped development version (51KB).

Previous versions

  • minified gzip format (version 0.0.1)
  • development version (version 0.0.1)

Readium.js

For the time being, Readium.js is available as both a zipped set of .js module files (54KB), and from source on Github. An approach for production releases still needs to be determined.

Previous versions

  • zipped set of .js module files (version 0.0.1)
